# Superstore Sales Analysis

## Overview
Sales analysis project using SQL and Power BI on the Superstore dataset (9,994 orders, $2.3M total sales).

## Tools Used
- SQL (SQLite) — data querying and analysis
- Power BI Desktop — interactive dashboard
- GitHub — version control and portfolio

## Key Findings
- **Technology** is the highest revenue category ($836K) with the best profit margin
- **Furniture** has high sales but very low profit ($8.70 avg) — discount issue
- **West region** is the most profitable ($108K profit)
- **Cubify CubeX 3D Printer** is the biggest loss-maker (-$8,879)
- Total of 38K units sold across 4 regions

## SQL Queries
Key analyses performed:
- Total sales, orders and average order value
- Sales and profit by category
- Top 10 most profitable products
- Top 10 loss-making products
- Sales and profit by region
- Monthly sales trend (2014–2017)

## Dashboard
Power BI dashboard includes:
- KPI Cards: Total Sales, Total Profit, Total Quantity
- Bar chart: Sales by Category
- Bar chart: Profit by Region
- Line chart: Sales trend over time
